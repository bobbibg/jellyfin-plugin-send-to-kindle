SMTP Server (admin only)

Configure the email account used to send books. The "From" address must be added to your Amazon approved-senders list at amazon.com/sendtokindle, otherwise Amazon will silently drop the messages.

e.g. mail.gmx.com or smtp.gmail.com
587 for STARTTLS (recommended), 465 for implicit SSL.
For GMX, generate a dedicated mail-app password in your GMX security settings.
Must be in your Amazon approved-senders list.
Amazon's documented cap is 50 MB per email.
